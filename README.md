# Unpopular Baby

[Start your search here.](https://www.unpopularbaby.com/)

### Full stack React web application to search for names using popularity by decade and/or letter placements.

## Uses: 

 * React.js
 * Chart.js
 * SQL
 * Sequelize
 * Node
 * Express
 * Axios
 * Javascript
 * HTML
 * CSS
 * Flexbox
 * FontAwesome
 * Google Fonts
 * Responsive Bootstrap design
